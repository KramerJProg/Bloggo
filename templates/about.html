About Me

Software Developer

My name is Kramer Jensen, and I am a software developer with a great passion for coding and working on large applications as well as solving complex issues. Learning is one of my strengths along with dedication to the product.

This my personal blog site project called Bloggo. It is basically to quickly showcase my skills for a full stack "from scratch" Python application utilizing the flask framework. This is also using a CSS template. This project has full CRUD functionality for an administrator as well as a UI for users that can register an account and login to their profile. They are also able to leave comments under the blogs in the comment section as well as deleting their own comments. Anonymous users who don't have an account can view the blogs, but they are unable to comment on the blogs.
